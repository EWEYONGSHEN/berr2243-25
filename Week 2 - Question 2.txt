Question 2: Identify all the mongo operators used in the exercise, then explain the usage for
each.

Answer:
MongoDB Operators Used:

1. $gte - "Greater Than or Equal"
   Example: rating: { $gte: 4.5 }
   Meaning: "Find drivers with at least 4.5 stars"

2. $inc - "Increment" 
   Example: { $inc: { rating: 0.1 } }
   Meaning: "Add 0.1 to the current rating"

Other MongoDB Methods:
- insertOne() - Add one driver
- find() - Search for drivers
- updateOne() - Update one driver
- deleteOne() - Remove one driver
